# Frula Homes — Web App

Nuxt 3 application powering [frulahomes.com](https://frulahomes.com).

See the [root README](../../README.md) for full project documentation.

## Development

```bash
# From the repo root:
npm run dev
```

## Build

```bash
npm run build
```

Deploys automatically to Cloudflare Pages on push to `main`.
